# Discord Bot System

## Overview
A simple Discord-style bot system with command handling and interactive messaging.

---

## Features
- Command-based interaction
- Custom bot responses
- Interactive terminal chat workflow

---

## Files
- `commands.py` → handles bot commands
- `bot.py` → runs bot interaction system

---

## Usage

```bash
python bot.py
```

---

## Example Commands

```text
!hello
!help
!ai
```

---

## Example Output

```text
User: !hello
Bot: Hello user!
```

---

## Architecture

```text
User Input
     ↓
Command Handler
     ↓
Bot Response
```
